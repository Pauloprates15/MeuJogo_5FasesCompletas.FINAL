# Meu Jogo - 5 Fases Completas

Este é um jogo de plataforma 2D desenvolvido na Unity como parte do projeto de Extensão Curricular. O jogo desafia o jogador a superar 5 fases completas com diferentes obstáculos, inimigos e mecânicas de progressão.

---

## 🎮 Como Jogar (Controles)

Use os seguintes comandos para controlar o personagem no teclado:

*   **Mover para a Esquerda**: Tecla `A` ou `Seta para Esquerda`
*   **Mover para a Direita**: Tecla `D` ou `Seta para Direita`
*   **Pular**: Tecla `Espaço` ou `Seta para Cima`
*   **Reiniciar Fase (se travar)**: Tecla `R`
*   **Pausar o Jogo**: Tecla `Esc`

---

## ⚙️ Mecânicas do Jogo

O jogo conta com mecânicas clássicas de plataforma projetadas para desafiar a habilidade do jogador:

*   **Movimentação Fluida**: Física customizada de corrida, aceleração e pulo preciso.
*   **Coletáveis**: Moedas espalhadas pelas fases 
*   **Sistema de Vida/Dano**: O jogador possui corações ou pontos de vida. Colidir com inimigos ou espinhos causa dano.
*   **Progressão de Fases**: O jogador precisa encontrar o portal de saída ou coletar um item chave para abrir a passagem e avançar para o próximo nível.
*   **5 Fases Únicas**: Cada fase introduz um layout diferente, maior dificuldade e posicionamento estratégico de inimigos.
